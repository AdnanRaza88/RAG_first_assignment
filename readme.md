# PDF RAG Chatbot 📄

Chat with your PDF using Groq and RAG.

## Features
- Upload any PDF
- Fast answers with Groq Llama 3.1
- Free embeddings with HuggingFace
- Shows source chunks

## How to Run Locally
1. pip install -r requirements.txt
2. streamlit run app.py

## Deploy on Streamlit Cloud
1. Push this repo to GitHub
2. Go to share.streamlit.io
3. Add GROQ_API_KEY in Secrets
4. Deploy

## Tech Stack
Groq + LangChain + FAISS + Streamlit
